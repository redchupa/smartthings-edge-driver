# 풍경채 SmartThings Edge 드라이버

CVnet 조명, 난방, 환풍기, 가스밸브, 날씨를 SmartThings 앱에서 제어 할수 있게 해주는 엣지 드라이버 입니다.

본 스마트싱즈 엣지드라이버는 

스마트싱즈 허브가 있으신 분만 사용이 가능합니다.
그리고 개인 집에서 개인 PC를 24시간 키고 (별도로 배포한) 윈도우 프로그램이 PC에서 항상 실행중이여야 정상 동작합니다.

해당 프로그램은 CVnet 및 아파트너 앱의 기능을 기반으로 편의성을 개선한 윈도우 실행형 프로그램 입니다. 복잡한 RS485 컨트롤러 설치 없이 단순히 로그인만으로 연동 해주는 중계 브릿지 프로그램이기에 로컬에서만 제공되는 엘리베이터 호출, 공동현관문 열기 기능은 제공되지 않습니다. HomeAssistant 환경에서도 사용자가 쉽게 연동하고 활용할 수 있도록 개발 완료한 상태이며, 스마트싱즈 앱에서 쉽게 사용하실수 있도록 엣지드라이버를 개발하여 공유 드립니다.

## 사용 방법

### 1. 초대 링크로 드라이버 추가

아래 링크를 클릭 → **ACCEPT** → 삼성 계정 로그인

**초대 링크**: https://bestow-regional.api.smartthings.com/invite/kVlnzLB0bol4

### 2. 브릿지 실행

PC에서 브릿지 프로그램(exe)을 실행하세요. (별도 배포)

- 기본적으로 포트 8098 사용
- PC IP 확인: `ipconfig` (예: 192.168.1.84)
- 브라우저에서 `http://PC_IP:8098/` 접속 시 "연결됨" 표시되면 정상

### 3. SmartThings 앱에서 기기 추가

1. **기기 추가** → **주변 기기 스캔**
2. 풍경채 조명, 난방, 환풍기 등이 나타나면 추가
3. 각 기기 **설정(⚙️)** 에서 입력:
   - **브릿지 주소**: PC의 IP
   - **브릿지 포트**: 8098
   - **GS Jeil 아이디** / **비밀번호**: CVnet 로그인 정보

## 지원 기기

| 기기 | 기능 |
|------|------|
| 조명 1~4 | on/off |
| 난방 1~5 | 온도 설정, on/off |
| 환풍기 | on/off |
| 가스밸브 | 원격 닫기 |
| 날씨 | 기온, 날씨, 미세먼지 |
