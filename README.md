# 풍경채 SmartThings Edge 드라이버

CVnet 조명, 난방, 환풍기, 가스밸브, 날씨를 SmartThings에서 제어합니다.  
집 안 PC에서 exe 프로그램을 실행하고 항상 PC를 켜두고 사용한다면 스마트싱즈 앱에서 제어 하실 수 있습니다.

---

## 사용 방법

### 1. 초대 링크로 드라이버 추가

아래 링크를 클릭 → **ACCEPT** → 삼성 계정 로그인

**초대 링크**: https://bestow-regional.api.smartthings.com/invite/kVlnzLB0bol4

### 2. 브릿지 실행

PC에서 브릿지 프로그램(exe)을 실행하세요. (별도 배포)

- 포트 8098 사용
- PC IP 확인: `ipconfig` (예: 192.168.1.84)
- 브라우저에서 `http://PC_IP:8098/` 접속 시 "연결됨" 표시되면 정상

### 3. SmartThings 앱에서 기기 추가

1. **기기 추가** → **주변 기기 스캔**
2. 풍경채 조명, 난방, 환풍기 등이 나타나면 추가
3. 각 기기 **설정(⚙️)** 에서 입력:
   - **브릿지 주소**: PC IP
   - **브릿지 포트**: 8098
   - **GS Jeil 아이디** / **비밀번호**: CVnet 로그인 정보

---

## 지원 기기

| 기기 | 기능 |
|------|------|
| 조명 1~4 | on/off |
| 난방 1~5 | 온도 설정, on/off |
| 환풍기 | on/off |
| 가스밸브 | 원격 닫기 |
| 날씨 | 기온, 날씨, 미세먼지 |
